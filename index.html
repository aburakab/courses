Programming Concepts

AI-generated study page for beginners

Learn the building blocks of programming with short definitions and examples.

Variables

Variables store information so your program can reuse and update values.

name = "Sam"
score = 95

Conditionals

Conditionals let your program choose actions based on true/false checks.

if score >= 60:
    print("Pass")
else:
    print("Try again")

Loops

Loops repeat instructions and are useful for processing lists or ranges.

for i in range(3):
    print(i)

Functions

Functions group reusable logic into named blocks.

def greet(user):
    return f"Hello, {user}!"
